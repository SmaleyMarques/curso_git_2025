## **Atalhos**  

- `Tab` → Completa palavras automaticamente.  
- `Ctrl + L` → Limpa a tela do terminal.  
- `↑` → Navega pelos comandos utilizados anteriormente.  
- `q` → Destrava o terminal quando congelado em `*:`.  
- **No Nano:**  
  - `Ctrl + O` → Salva o arquivo.  
  - `Ctrl + X` → Fecha o editor.
